# Platform Documentation

Welcome to the Internal Developer Platform documentation.

## What is this platform?

An enterprise-grade platform for accelerating service and ML delivery with:

- **Self-service catalog** for services and ML models
- **Golden paths** for standardized delivery
- **Built-in governance** with security and compliance
- **Unified observability** across all workloads
